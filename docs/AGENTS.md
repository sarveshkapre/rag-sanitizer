# AGENTS

## Working agreements
- Follow `docs/PLAN.md` and `docs/PROJECT.md` for architecture and commands.
- Keep changes minimal and focused; update tests and `docs/CHANGELOG.md` for user-facing changes.
- Avoid heavy dependencies; keep sanitizer deterministic and transparent.

## Commands
- Setup: `make setup`
- Dev: `make dev`
- Test: `make test`
- Lint: `make lint`
- Typecheck: `make typecheck`
- Build: `make build`
- Quality gate: `make check`
